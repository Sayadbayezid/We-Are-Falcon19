# 🦅 Falcon 19 - Official Team Portal

![Falcon 19 Banner](https://i.ibb.co/v4t8GJbM/IMG-3336.jpg)

Welcome to the official squad portal of **Falcon 19**, the SSC 2019 Batch Football Team of **Karimun Nessa Siddiqui High School**. This website serves as the digital home for our team, featuring player profiles, jersey details, and a dynamic rating system.

🌍 **Live Website:** [sayadbayezid.github.io/We-Are-Falcon19](https://sayadbayezid.github.io/We-Are-Falcon19/)

---

## ⚡ Features
* **Single Page Application (SPA):** Fast loading without page refreshes.
* **Dynamic Player Profiles:** Shareable target links for every individual player.
* **Interactive Modal System:** View details and rate players seamlessly.
* **Auto-Deployment:** CI/CD pipeline integrated using GitHub Actions.
